# Switzerland 2026 Luxury Travel Guide

Public travel guide and trip-operations site for a 2026 Switzerland itinerary.

## GitHub Pages

Use the repository root as the GitHub Pages source. The public entry point is:

```text
https://YOUR_GITHUB_USERNAME.github.io/Switzerland_2026/
```

Replace `YOUR_GITHUB_USERNAME` with the GitHub account name after deployment.
